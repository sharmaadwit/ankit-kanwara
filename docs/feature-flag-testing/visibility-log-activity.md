# Testing: Log activity shortcut (`logActivity`)

## What it controls

Quick **Log Activity** entry points that are tagged with `data-dashboard="logActivity"` (e.g. primary dashboard button in card dashboard HTML).

## Enable

Matrix row **Log Activity Shortcut** ON.

## Verify (on)

- Dashboard (and other surfaces using this key) show the quick log button.

## Verify (off)

- Those controls are hidden (`dashboard-hidden`).

## Note

Does not necessarily remove logging from every context—only elements wired to this visibility key.
